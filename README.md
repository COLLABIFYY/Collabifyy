# Collabifyy

Collabifyy is a platform connecting Brands and Creators for seamless collaboration.

## Features
- **Smart Discovery:** AI-powered matching for brands and creators.
- **Campaign Tracking:** Real-time analytics and updates.
- **Secure Payments:** Escrow-backed payments for creators.
- **Collab Agreement:** Digital contracts for safe partnerships.

## Tech Stack
- React + Vite
- Supabase (Auth & Database)
- Lucide React (Icons)
- CSS (Custom styling)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```
